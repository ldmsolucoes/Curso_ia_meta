********************************************** COMO RODAR O PROJETO **********************************************

1. Clonar o projeto do GitHub
2. Criar um ambiente virtual (.venv) na raiz do projeto
3. Instalar os pacotes via arquivo "requirements.txt" no ambiente virtual
4. Criar uma pasta na raiz com o nome "NFs_zip" e colar o arquivo "202401_NFs.zip" nela
5. Rodar o arquivo "nfe.py"

Aparecerá um menu com três opções:

Opção 1: Descompacta o .zip para a pasta "NFs_Extraidas" localizada na raiz do projeto
Opção 2: Cria o banco de dados vetorial e prepara os CSVs para uso
Opção 3: Pesquisa as notas conforme indicado no prompt
   3.1 Os dados sobre as notas exibidas do prompts são dados válidos da base de csvs
   3.1 A pesquisa por semântica da base sempre é chamada quando nada é encontrado(fallback)

Boa Consulta!!!
